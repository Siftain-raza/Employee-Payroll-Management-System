## 📖 Overview

The **Employee Payroll Management System** is a Linux Shell Scripting project developed to automate employee record management and payroll processing.

The system provides a menu-driven interface that allows users to manage employee information, calculate salaries, generate payslips, and maintain payroll records efficiently using Bash shell scripting.

This project demonstrates the practical use of Linux shell scripting, file handling, conditional statements, loops, and command-line utilities for automating real-world payroll operations.

---

## ✨ Features

- 👤 Add new employee records
- 📝 Update employee information
- 🔍 Search employee details
- ❌ Delete employee records
- 💰 Calculate employee salary
- 📄 Generate employee payslip
- 📂 Store employee details in text files
- 📋 Menu-driven user interface
- ⚡ Fast payroll processing using Shell Scripts

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Siftain-raza/Employee-Payroll-Management-System.git
```

### Navigate to the project folder

```bash
cd Employee-Payroll-Management-System
```

### Give execution permission

```bash
chmod +x *.sh
```

### Run the project

```bash
./Main_Menu.sh
```
